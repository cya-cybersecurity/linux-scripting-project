# linux-scripting-project
Linux-based calorie calculator application developed using Bash, Python, Perl, and AWK.

## Overview
This project is a Linux-based calorie calculator application designed to help users estimate calorie intake goals based on BMI, current weight, and target weight-loss objectives.

## Technologies Used
- Bash
- Python
- Perl
- AWK
- Linux

## Features
- User input collection
- BMI-based calorie calculations
- Personalized calorie recommendations
- Data processing and automation using scripting languages

## Skills Demonstrated
- Linux scripting
- Command-line application development
- Data processing
- Automation
- Problem solving
- Input validation

## Project Objectives
The goal of this project was to develop scripting and automation skills in a Linux environment while building a functional application capable of processing user data and generating nutritional recommendations.

## Lessons Learned
This project strengthened my understanding of Linux environments, scripting logic, automation workflows, and integrating multiple scripting languages to accomplish application tasks.
